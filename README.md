# To-do List App 📝

Mini proyecto web en HTML, CSS y JavaScript para crear una lista de tareas.

## Funciones

- Añadir tareas
- Ver lista en pantalla
- Código simple y ordenado

## Cómo usar

1. Clona el repositorio
2. Abre `index.html` en tu navegador
